# Conceptual Explanation Documents

This folder contains conceptual explanation documents suitable for presentations and educational purposes.

## Purpose

These documents are intended to:
- **Explain technical concepts** in an accessible way
- **Provide presentation material** for demos and explanations
- **Include visual diagrams** (ASCII art) that are easy to present
- **Use analogies** to make complex concepts understandable

## Documents

- **[retry-logic-explanation.md](./retry-logic-explanation.md)** - Explanation of why retry logic is needed and how it works
- **[workers-overview.md](./workers-overview.md)** - Complete overview of all three workers (Indexer, Resonance, Reteller) and how they collaborate

## Usage

These documents can be used for:
- Presentations to stakeholders
- Onboarding new team members
- Educational purposes
- User documentation

## Format

All documents use:
- **ASCII art diagrams** for visual explanations
- **Clear examples** with emojis for engagement
- **Analogies** from everyday life
- **Key points** for presentations
